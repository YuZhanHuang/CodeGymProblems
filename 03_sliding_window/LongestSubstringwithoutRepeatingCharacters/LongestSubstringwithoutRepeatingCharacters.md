# Longest Substring without Repeating Characters

## Statement

Given a string, input_str, return the length of the longest substring without repeating characters.

## Example:

input_str = "abcabcbb" → 3 "abc"

input_str = "abcdef" → 6 "abcdef"

input_str = "z" → 1 "z"

input_str = "zz" → 1 "z"
