# Swapping Nodes in a Linked List

## Statement

Given the linked list and an integer, k, return the head of the linked list after swapping the values of the kth node from the beginning and the kth node from the end of the linked list.

**Note:** We'll number the nodes of the linked list starting from 1 to n.

## LeetCode Link

https://leetcode.com/problems/swapping-nodes-in-a-linked-list/description/?utm_source=chatgpt.com

## Constraints

The linked list will have n number of nodes.

- 1 ≤ k ≤ n ≤ 500
- −5000 ≤ Node.value ≤ 5000
